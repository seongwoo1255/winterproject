어텐션 레이어의 순전파(forward) 계산.

Query, Key, Value를 기반으로 어텐션 스코어를 계산하고, 어텐션 확률과 문맥 벡터(context vector)를 생성합니다.

Parameters
hidden_states입력 숨겨진 상태 텐서 (batch_size x seq_length x hidden_size).
attention_mask어텐션 마스크 (batch_size x seq_length x seq_length). 패딩 토큰에 대한 마스킹을 처리.
head_mask특정 어텐션 헤드를 마스킹할 때 사용 (optional).
encoder_hidden_states크로스 어텐션일 경우 인코더의 출력 상태.
encoder_attention_mask크로스 어텐션일 경우 인코더 마스크.
past_key_value이전 단계에서 계산된 Key와 Value (optional).
output_attentions어텐션 확률을 출력할지 여부 (default: False).
Returns
문맥 벡터, 어텐션 확률 (optional), 그리고 past_key_value.
