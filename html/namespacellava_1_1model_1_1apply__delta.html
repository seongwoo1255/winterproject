Delta를 기존 모델에 적용하여 새로운 모델을 생성하는 함수

Author
이지희(2021113406)
  • Delta는 Base 모델의 파라미터에 더해져 Target 모델이 생성됩니다.
  • Delta 및 Base 모델의 파라미터가 일치하지 않을 경우 예외가 발생합니다.
Parameters
base_model_path(str) Base 모델의 경로
target_model_path(str) 생성될 Target 모델의 저장 경로
delta_path(str) Delta 모델의 경로
Exceptions
AssertionErrorDelta와 Base 모델 간 매칭되지 않는 파라미터가 있을 경우 발생
