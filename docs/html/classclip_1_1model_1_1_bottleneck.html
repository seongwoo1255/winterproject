Detailed Description

Implements a ResNet bottleneck block with optional downsampling.

This class includes three convolutional layers with BatchNorm and ReLU activation. An optional downsampling layer is added when the stride is greater than 1 or the input and output channels differ.

Parameters
inplanesNumber of input channels.
planesNumber of output channels.
strideStride for the second convolution.
