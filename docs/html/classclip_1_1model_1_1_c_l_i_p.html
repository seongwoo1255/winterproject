Detailed Description

Implements the CLIP model, combining image and text encoders.

Encodes image and text inputs into a shared embedding space, enabling multimodal understanding.

Parameters
embed_dimDimensionality of the shared embeddings.
image_resolutionResolution of input images.
vision_layersSpecification of the vision encoder's layers.
vision_widthWidth of the vision encoder.
vision_patch_sizePatch size for the Vision Transformer (if used).
context_lengthMaximum context length for text inputs.
vocab_sizeSize of the text vocabulary.
transformer_widthDimensionality of the text embeddings.
transformer_headsNumber of attention heads in the text encoder.
transformer_layersNumber of layers in the text encoder.
