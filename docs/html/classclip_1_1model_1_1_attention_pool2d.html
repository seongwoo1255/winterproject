Detailed Description

Implements a 2D attention pooling layer.

This class includes positional embeddings and multi-head self-attention for pooling spatial features.

Parameters
spacial_dimThe spatial dimension of the input feature map.
embed_dimDimensionality of the feature embeddings.
num_headsNumber of attention heads.
output_dimDimensionality of the output embeddings (default: same as embed_dim).
