Detailed Description

Implements a Vision Transformer model.

Encodes input images into a sequence of patch embeddings and processes them using a Transformer.

Parameters
input_resolutionInput image resolution.
patch_sizeSize of image patches.
widthDimensionality of the embeddings.
layersNumber of Transformer layers.
headsNumber of attention heads.
output_dimDimensionality of the output embeddings.
