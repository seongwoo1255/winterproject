Detailed Description

LLaVA MPT 변형을 통한 Causal Language Model 구현

Author
이지희(2021113406)
  • 멀티모달 입력 데이터를 기반으로 텍스트를 생성하는 모델입니다.
  • MPT 모델과 Vision Encoder의 출력(Hᵥ)을 결합하여 멀티모달 입력(Hₐ)을 생성합니다.
  • 생성된 멀티모달 입력을 바탕으로 텍스트 응답(Xₐ)을 생성합니다.
