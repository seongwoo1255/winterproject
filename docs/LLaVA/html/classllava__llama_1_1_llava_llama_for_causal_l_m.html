멀티모달 Causal Language Model의 forward 패스를 정의합니다.

Author
이지희(2021113406)

이 메서드는 멀티모달 입력 데이터를 모델에 전달하여 예측 결과를 반환합니다. 텍스트 입력과 이미지 입력을 결합하여 멀티모달 데이터를 처리합니다. 이미지 특징 벡터(Hᵥ)와 텍스트 특징 벡터(Hₓₓ)를 통합한 멀티모달 입력(Hₐ)을 생성하고, 이를 기반으로 Causal Language Model의 출력을 생성합니다.

Parameters
input_ids(torch.LongTensor) 텍스트 입력 토큰 ID (optional)
attention_mask(torch.Tensor) 입력에 대한 어텐션 마스크 (optional)
position_ids(torch.LongTensor) 입력의 위치 정보 ID. (optional)
past_key_values(List[torch.FloatTensor]) 이전 키/값 캐시. 디코더에 사용 (optional)
inputs_embeds(torch.FloatTensor) 입력 임베딩 (optional)
labels(torch.LongTensor) 정답 레이블 (optional)
use_cache(bool) 이전 키/값 캐시를 사용할지 여부 (optional)
output_attentions(bool) 어텐션 가중치를 출력할지 여부 (optional)
output_hidden_states(bool) 히든 상태를 출력할지 여부 (optional)
images(torch.FloatTensor) 입력 이미지 텐서 (optional)
image_sizes(List[List[int]]) 입력 이미지 크기 정보 (optional)
return_dict(bool) 반환 값을 딕셔너리 형식으로 출력할지 여부 (optional)
Returns
  • Union[Tuple, CausalLMOutputWithPast]: 모델의 출력. 기본적으로 CausalLMOutputWithPast 형식으로 반환됩니다.
  • 텍스트 출력과 필요한 경우 어텐션 정보 및 히든 상태를 포함합니다.
Exceptions

None. 모든 입력은 optional로 설계되어 있으며, 적절한 처리 과정을 통해 오류를 방지합니다.

Author

이 메서드는 텍스트와 이미지 입력을 결합하여 텍스트 생성 작업을 수행합니다. LLaVA 모델의 언어 생성 기능을 사용하며, 멀티모달 입력이 제공된 경우 적절히 처리하여 최종 응답 텍스트를 반환합니다.

Parameters

images(torch.Tensor) 입력 이미지 데이터 (optional)
image_sizes(torch.Tensor) 이미지 크기 정보 (optional)
**kwargs추가 매개변수. 모델의 생성 작업에 필요한 설정 정보를 포함할 수 있습니다
Returns
  • Union[GenerateOutput, torch.LongTensor]: 생성된 텍스트 결과
  • 모델에 따라 텍스트 출력과 관련 정보를 포함합니다.
Exceptions
