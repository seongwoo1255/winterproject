Detailed Description

LLaVA 모델의 추상 클래스

멀티모달 입력 데이터를 처리하고 텍스트 모델과 비전 타워를 연결하기 위한 기본 기능을 제공합니다. 이 클래스는 추상 클래스(ABC)로, 구체적인 구현은 상속받는 클래스에서 정의해야 합니다.

Author
이지희(2021113406)
  • 비전 타워를 사용하여 이미지 특징을 추출한다.
  • 멀티모달 입력(텍스트와 이미지)을 처리하여 모델 학습에 적합한 형태로 변환한다.
  • 즉, Vision Encoder 출력(Zᵥ)부터 멀티모달 데이터 준비까지의 역할을 담당한다.
  • 특히, 이미지와 텍스트 데이터를 통합하여 언어 모델 입력으로 준비하는 중간 과정을 처리한다.
