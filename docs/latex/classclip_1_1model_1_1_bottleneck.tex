\doxysubsection{Detailed Description}
Implements a Res\+Net bottleneck block with optional downsampling. 

This class includes three convolutional layers with Batch\+Norm and Re\+LU activation. An optional downsampling layer is added when the stride is greater than 1 or the input and output channels differ.
