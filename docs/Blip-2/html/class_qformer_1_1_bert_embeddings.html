입력 데이터를 임베딩 벡터로 변환하는 함수.

입력 토큰과 위치 정보를 사용하여 임베딩 벡터를 생성하고, 필요 시 질의 임베딩(query embeddings)을 결합합니다.

Parameters
input_ids입력 토큰 ID (batch_size x seq_length).
position_ids위치 ID (batch_size x seq_length). 기본값은 None으로 설정되며 자동 생성됩니다.
query_embeds추가로 결합할 질의 임베딩 (query embeddings).
past_key_values_length과거 시퀀스 길이 (position ID 계산 시 사용).
Returns
임베딩 벡터 (batch_size x seq_length x hidden_size).
